# 🇩🇪 2025 德國南部親子自駕・最終定案版 (The Perfect Loop)

## 基本資訊
- **總天數**：12 天 11 夜
- **成員**：2 大 1 小 (11歲)
- **住宿策略**：5 個基地 (羅騰堡、慕尼黑、薩爾斯堡、嘉米許、海德堡)
- **關鍵點**：逆時針環島、不走回頭路、週日避開購物。

## 📅 行程詳情

### 第一階段：童話小鎮與中世紀 (Base 1: 羅騰堡)
- **D1 (5/29 五)**：FRA 機場取車 ➔ 烏茲堡 (主教宮、舊緬茵橋) ➔ 羅騰堡 (宿)
- **D2 (5/30 六)**：羅騰堡全日 (城牆漫步、聖誕博物館、守夜人導覽) ➔ (宿羅騰堡)
  - *提醒：超市補給日！明後兩天商店不開。*

### 第二階段：巴伐利亞首府 (Base 2: 慕尼黑)
- **D3 (5/31 日)**：羅騰堡 ➔ 慕尼黑 (英國花園衝浪、公園跑跳) ➔ (宿慕尼黑)
  - *提醒：週日商店全關。*
- **D4 (6/1 一)**：BMW Welt (展示區)、奧林匹克公園 ➔ 瑪麗恩廣場、皇家啤酒屋 ➔ (宿慕尼黑)
  - *提醒：國定假日，商店休息。*

### 第三階段：湖區與莫札特 (Base 3: 薩爾斯堡)
- **D5 (6/2 二)**：慕尼黑 ➔ 國王湖 (紅蔥頭教堂) ➔ 薩爾斯堡 (宿)

### 第四階段：阿爾卑斯山巔 (Base 4: 嘉米許)
- **D6 (6/3 三)**：薩爾斯堡 (米拉貝爾花園) ➔ 阿爾卑斯街道 B305 景觀公路 ➔ 嘉米許 (宿)
- **D7 (6/4 四)**：楚格峰 (登頂德國最高峰) ➔ 艾布湖 (玩水) ➔ (宿嘉米許)
- **D8 (6/5 五)**：新天鵝堡 (瑪麗恩橋) ➔ 威斯教堂 ➔ (宿嘉米許)
  - *提醒：務必 6 週前搶票。*

### 第五階段：汽車、古堡與萊茵河 (Base 5: 海德堡)
- **D9 (6/6 六)**：嘉米許 ➔ 斯圖加特 (賓士博物館) ➔ 海德堡 (宿)
- **D10 (6/7 日)**：海德堡全日 (古堡纜車、哲學家之道、無車日) ➔ (宿海德堡)
- **D11 (6/8 一)**：海德堡 ➔ 呂德斯海姆 (萊茵河、葡萄園纜車) ➔ 返回海德堡 (宿)

### 完美收尾
- **D12 (6/9 二)**：海德堡 ➔ FRA 機場還車、退稅、搭機
